% Beyond the default SparseLDA sweep, \pkg{topica} exposes the inference engine
% of each model as a pluggable backend behind one fitted-model contract,
% selected with \code{sampler} (count-based models) or \code{inference}
% (variational models). For the Dirichlet-prior family, \code{"warp"} is the
% cache-efficient WarpLDA sampler \citep{chen2016warplda}, an $O(1)$-per-token
% Metropolis--Hastings scheme whose per-sweep cost is flat in $K$; at the
% fine-grained topic counts common in social science (e.g.\ $K=1{,}000$) it fits
% several times faster than SparseLDA and reaches higher coherence.
% \code{"cvb0"} is collapsed variational Bayes, zeroth-order
% \citep{asuncion2009smoothing}, a deterministic, non-sampling alternative that
% trades speed for higher topic coherence and exact reproducibility. These
% extend across the count-based models that share the collapsed conditional:
% the per-document prior (\code{DMR}), seeded (\code{SeededLDA}), supervised
% (\code{LabeledLDA}), and keyword (\code{keyATM}) variants. CVB0 is the natural
% home for the supervised and keyword models in particular, because its
% per-cell responsibilities take the label mask or keyword-switch constraint
% exactly and at no cost (the off-mask responsibility is simply zero), where a
% masked Metropolis--Hastings proposal would barely mix.
%
% The logistic-normal models have no Dirichlet to collapse, so \code{CTM} (the
% engine \code{STM} builds on) instead exposes stochastic variational inference
% \citep{hoffman2013svi} through \code{inference="svi"}: minibatch Laplace
% E-steps drive Robbins--Monro updates of the global topics, mean, and
% covariance, so one epoch touches every document while the global state stays
% minibatch-sized. It is the backend for corpora too large to sweep in full
% each EM step, with the full-batch variational EM remaining the default.
